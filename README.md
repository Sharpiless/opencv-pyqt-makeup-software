# 基于opencv+pyqt的美颜软件

## 安装依赖：

```bash
pip install dlib
```
```bash
pip install opencv-python
```
```bash
pip install pyqt5
```
## 下载权重：
需要dlib的68关键点检测权重，下载地址：

[http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2](http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2)

下载完成后放到data文件夹下。

## 运行代码：

```bash
python MakupGUI.py
```
